How does this work?

This tool simulates a ping by making a HEAD request to the target server or attempting to load a resource from it. It measures the "Time to First Byte" (TTFB) or completion time.

Note: Due to browser security policies (CORS), exact ICMP ping is not possible from a web page. This tool provides a "HTTP Latency" estimation which is close to real-world loading performance.

Popular Targets: Google, Cloudflare, Discord, GitHub.
