# 🚀 كيفية نشر موقع DGToolbox على GitHub

بما أنني لا أستطيع الدخول إلى حساب GitHub الخاص بك مباشرة، فقد قمت بتجهيز كل الملفات اللازمة. عليك فقط اتباع هذه الخطوات البسيطة لرفع الموقع وتشغيله أونلاين مجاناً.

## الخطوة 1: إنشاء مستودع (Repository) جديد
1. اذهب إلى [GitHub.com](https://github.com) وسجل الدخول.
2. اضغط على علامة **+** في الزاوية العلوية اليمنى واختر **New repository**.
3. اكتب اسم المستودع (مثلاً: `dg-toolbox`).
4. تأكد من اختيار **Public**.
5. **لا** تقم بإضافة README أو .gitignore أو License (لقد قمنا بذلك بالفعل).
6. اضغط على **Create repository**.

## الخطوة 2: رفع الملفات من جهازك
افتح التيرمينال (Terminal) أو موجه الأوامر (CMD) في مجلد المشروع، ثم انسخ والصق الأوامر التالية واحداً تلو الآخر:

```bash
# 1. تهيئة الجيت (إذا لم يكن مفعلاً)
git init

# 2. إضافة جميع الملفات
git add .

# 3. حفظ التغييرات
git commit -m "First commit: DGToolbox V1"

# 4. تغيير الفرع إلى main
git branch -M main

# 5. ربط المشروع بالمستودع الذي أنشأته
# استبدل الرابط أدناه برابط المستودع الخاص بك من الخطوة 1
git remote add origin https://github.com/YOUR_USERNAME/dg-toolbox.git

# 6. رفع الملفات
git push -u origin main
```

*(سيطلب منك تسجيل الدخول إذا كانت هذه أول مرة)*

## الخطوة 3: تفعيل GitHub Pages (لجعل الموقع يعمل أونلاين)
1. اذهب إلى صفحة المستودع الخاص بك على GitHub.
2. اضغط على **Settings** (الإعدادات) من الشريط العلوي.
3. من القائمة الجانبية اليسرى، اختر **Pages**.
4. تحت قسم **Build and deployment**، ومن خيار **Branch**، اختر `main` ثم اضغط **Save**.
5. انتظر دقيقة أو دقيقتين، وستظهر لك رسالة في الأعلى برابط موقعك الجديد (مثلاً: `https://your-username.github.io/dg-toolbox/`).
