## 🌟 Overview

**DevGamerToolbox** is a collection of essential tools for developers and gamers, built with privacy and performance in mind. Unlike other online tools, this project runs **100% client-side**. This means your data (passwords, code, images) **never** leaves your browser.

## ✨ Key Features

*   **🚀 100% Client-Side:** All calculations and processing happen locally in your browser.
*   **🔒 Privacy First:** No tracking, no analytics, no server-side storage.
*   **📱 PWA Ready:** Installable as a native app on Windows, Android, and iOS.
*   **🌍 Multilingual:** Fully localized support for **English** and **Arabic (RTL)**.
*   **🎨 Premium UI:** Modern glassmorphism design with customizable themes (Dark, Amoled, Neon, etc.).
*   **⚡ Offline Capable:** Works perfectly without an internet connection using Service Workers.

## 🧰 Tools Included

| Category | Tool | Description |
| :--- | :--- | :--- |
| **Dev** | **JSON Formatter** | Validate, format, minify, and view JSON trees. |
| **Dev** | **Regex Tester** | Real-time regular expression testing and replacement. |
| **Dev** | **Markdown Editor** | Write Markdown with live preview and HTML export. |
| **Dev** | **Diff Checker** | Compare two text versions to spot changes or bugs. |
| **Dev** | **Base64 Converter** | Encode and decode text or files to Base64 strings. |
| **Dev** | **UUID Generator** | Generate random UUIDs (v4) in bulk. |
| **Security** | **Password Generator** | Create strong, secure passwords with custom rules. |
| **Security** | **Text Encrypt (AES)** | Encrypt confidential text using AES-256 directly in the browser. |
| **Media** | **Image Compressor** | Compress and resize images to save space without quality loss. |
| **Media** | **QR Code Generator** | Generate custom QR codes for URLs, text, and Wi-Fi. |
| **Design** | **Color Picker Pro** | Advanced color selection with history and format conversion (HEX, RGB). |
| **Utility** | **Unit Converter** | Convert between common units (length, weight, digital storage). |
| **Network** | **Ping Checker** | Check the response time (latency) of websites. |

2.  Navigate to the project folder:

## 💻 Tech Stack

*   **Core:** HTML5, CSS3 (Vanilla), JavaScript (ES6+).
*   **Libraries:**
    *   `FontAwesome` (Icons)
    *   `Marked.js` (Markdown Parsing)
    *   `Highlight.js` (Syntax Highlighting)
    *   `JSZip` / `Browser-Image-Compression` (Media Tools)
    *   `Diff.js` (Text Comparison)
